# Проверка работоспособности виджета

## Быстрая проверка

1. **Создайте тестовый HTML файл:**
```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Тест виджета</title>
</head>
<body>
    <h1>Тестовая страница</h1>
    <p>Виджет должен появиться в правом нижнем углу</p>
    
    <!-- Виджет -->
    <script src="https://ai-studia.ru/widget.js?site=test.ru" async></script>
</body>
</html>
```

2. **Откройте в браузере и проверьте консоль (F12):**
   - Должно быть: `AI Widget загружен и инициализирован`
   - Должно быть: `Video Widget инициализирован.`
   - Не должно быть ошибок загрузки файлов

3. **Проверьте визуально:**
   - Виджет появляется в правом нижнем углу
   - При клике открывается меню с кнопками
   - Видео проигрывается

## Проверка загрузки файлов

Откройте вкладку Network (Сеть) в DevTools и проверьте, что загружаются:
- ✅ `widget.css` - статус 200
- ✅ `jquery-3.6.0.min.js` - статус 200
- ✅ `socket.io.min.js` - статус 200
- ✅ `script.js` - статус 200
- ✅ `avatar.mp4` - статус 200
- ✅ `orig.webp` - статус 200

## Возможные проблемы

### Виджет не появляется
- Проверьте консоль на ошибки
- Убедитесь, что все файлы доступны на `ai-studia.ru`
- Проверьте CORS настройки

### Ошибка "jQuery is not defined"
- jQuery загружается асинхронно, подождите немного
- Проверьте, что jQuery загрузился в Network

### Виджет появляется дважды
- Убедитесь, что скрипт подключен только один раз на странице

### Видео не проигрывается
- Проверьте, что `avatar.mp4` доступен
- Проверьте формат видео (должен быть MP4)

## Проверка на разных страницах

Виджет должен работать на любой странице, где подключен скрипт. Протестируйте:
- ✅ Главная страница
- ✅ Страница с большим количеством контента
- ✅ Мобильная версия (адаптивность)
- ✅ Разные браузеры (Chrome, Firefox, Safari)
